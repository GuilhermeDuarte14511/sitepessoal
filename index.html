Sistema de Agendamentos para Barbearia

Este projeto é um sistema de agendamentos desenvolvido para uma barbearia, permitindo que os usuários façam login utilizando seu email ou número de telefone. O sistema fornece um painel completo, onde os clientes podem visualizar seu histórico de serviços ou agendar novos. As funcionalidades incluem a seleção de serviços, escolha de barbeiros, visualização de horários disponíveis e um resumo detalhado com confirmação via email tanto para o cliente quanto para o barbeiro.

  • Login via email ou telefone
  • Visualização de histórico de serviços, incluindo valor total, barbeiro e status
  • Seleção e gerenciamento de serviços (adicionar/remover do carrinho)
  • Escolha de barbeiro com horários disponíveis com base na duração dos serviços
  • Confirmação de agendamento com envio de email contendo o resumo para o cliente e o barbeiro
